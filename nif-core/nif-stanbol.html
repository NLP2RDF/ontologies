Introduction

Ontology models

This ontology is split in three parts:

* a terminological model (containing rdfs:comment, rdfs:label, rdfs:subClassOf, rdfs:subPropertyOf, rdfs:range and rdfs:domain statements)

* an inference model (containing owl:TransitiveProperty, owl:hasKey)

* a validation model (containing owl:FunctionalProperty, owl:DisjointWith)

Feedback

If you'd like to leave feedback, please open an issue on GitHub (https://github.com/NLP2RDF/ontologies/issues) and read the README (https://github.com/NLP2RDF/ontologies#readme) or write an email to the mailing list: http://lists.informatik.uni-leipzig.de/mailman/listinfo/nlp2rdf

# Relation to the FISE ontology

FISE is an ontology originated from Apache Stanbol and found here: http://fise.iks-project.eu/ontology/

The ontology was compatible and useful and we just included it here.
